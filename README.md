## Run Locally

Clone the project

```bash
  git clone https://github.com/MOIN-AKHTAR/aws-passwordless-cognito.git
```

Go to the project directory

```bash
  cd aws-passwordless-cognito
```

Install dependencies

```bash
yarn
```

Run development server

```bash
yarn dev
```

Create production build

```bash
yarn run build
```

Prettify code

```bash
yarn lint:fix
```
